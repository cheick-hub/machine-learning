# Machine Learning Notes

This repository contains Markdown notes about machine learning.

The goal is to keep simple explanations of important ML concepts in one place.

## Topics

The notes can cover any area of machine learning, including:

- Supervised learning
- Unsupervised learning
- Semi-supervised learning
- Reinforcement learning
- Deep learning
- Optimization
- Probability and statistics
- Model evaluation
- Practical ML workflows

## Writing Notes

Each note should be clear and self-contained:

- Start with the main idea.
- Explain the intuition.
- Define important terms.
- Add formulas when they help.
- Use fenced `math` blocks for display formulas so GitHub renders them correctly.
- Add examples when useful.
